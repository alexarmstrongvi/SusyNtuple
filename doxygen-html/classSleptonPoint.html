Detailed Description

A slepton point in the model parameter space.

A slepton point is defined in the model parameter space by tree values: (1) the slepton mass (2) the m1 mass (3) the final state (left-handed or right-handed)

See this wiki for more information https://twiki.cern.ch/twiki/bin/viewauth/AtlasProtected/SUSYSignalGridDirectSlepton
