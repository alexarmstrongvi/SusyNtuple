Detailed Description

Trigger logic for dilepton events.

This class will implement the dilepton trigger logic based on the leptons, run number and stream The user will need to call one basic method: passDilTrig(Electrons,Muons,RunNumber,Stream) and it will return pass or fail based on kinematicsx
